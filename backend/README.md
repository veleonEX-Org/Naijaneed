# NaijaNeed Backend

Express.js API for the NaijaNeed platform.

## Setup

1. `npm install`
2. Create `.env` based on `.env.example`.
3. Fill in your Supabase URL and Service Role Key (for admin tasks).
4. Run migrations using the script in the artifacts.

## Scripts

- `npm run dev`: Start dev server with nodemon.
- `npm run build`: Compile TypeScript.
- `npm start`: Run compiled code.
